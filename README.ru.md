# Friends2GIS - Трекер местоположения друзей 2GIS

![Friends2GIS Icon](static/Friends2GIS.png)

Система отслеживания местоположения на основе WebSocket для мониторинга местоположения друзей из сервиса 2GIS Friends.

## Описание

Модуль `Friends2GIS` предоставляет возможности отслеживания местоположения для платформы osysHome путем подключения к WebSocket сервису 2GIS Friends. Он обеспечивает мониторинг и обновления местоположения в реальном времени.

## Основные возможности

- ✅ **Подключение WebSocket**: Подключение в реальном времени к сервису 2GIS Friends
- ✅ **Отслеживание местоположения**: Отслеживание местоположения друзей
- ✅ **Автоматические обновления**: Обновления местоположения в реальном времени
- ✅ **Переподключение**: Автоматическое переподключение при отключении
- ✅ **Расчет области просмотра**: Автоматический расчет области просмотра для отображения карты

## Панель администратора

Модуль предоставляет административный интерфейс для:
- Просмотра отслеживаемых друзей
- Управления местоположениями друзей
- Настройки подключения WebSocket

## Конфигурация

- **Токен**: Токен аутентификации 2GIS Friends
- **Интервал переподключения**: Задержка между попытками переподключения (по умолчанию: 5 секунд)

## Использование

### Настройка

1. Получите токен 2GIS Friends
2. Перейдите в модуль Friends2GIS
3. Введите токен в настройках
4. Модуль подключается автоматически

## Технические детали

- **Протокол**: WebSocket (WSS)
- **Сервис**: API 2GIS Friends
- **Многопоточность**: Отдельный поток для подключения WebSocket
- **Переподключение**: Обработка автоматического переподключения
