# Friends2GIS — руководство пользователя

## Назначение

Плагин **Friends2GIS** подключается к сервису **2GIS «Друзья»** по защищённому каналу **WebSocket** и получает координаты людей, с которыми у вас настроен обмен геолокацией в экосистеме 2GIS. Данные сохраняются в osysHome и при необходимости могут передаваться в модуль **GpsTracker** (если для пользователя включена соответствующая опция).

Используйте плагин, если вам нужно:

- видеть актуальное местоположение друзей из 2GIS на стороне умного дома;
- строить сценарии автоматизации по приближению/удалению (через связку с трекером и объектами системы);
- централизованно хранить историю обновлений в одной инсталляции osysHome.

## Как это работает

1. После ввода **токена** плагин открывает постоянное соединение с сервером  
   `wss://zond.api.2gis.ru/...` (API «зонда» 2GIS для пользовательских каналов).
2. Сервер присылает сообщения с состоянием друзей (в т.ч. координаты, батарею, время последней активности).
3. Плагин сопоставляет данные с записями в своей базе, обновляет поля и при включённой опции вызывает **GpsTracker**.
4. Интервал **«Минимальный интервал обновления (сек)»** ограничивает частоту записи, чтобы не перегружать систему при частых пакетах.

На странице настроек отображается **индикатор WebSocket** (цветной бейдж): подключено, подключение, переподключение, отключено, ошибка авторизации или отсутствие токена. Состояние периодически обновляется без перезагрузки страницы.

## Настройка в osysHome

1. Установите и включите модуль **Friends2GIS**.
2. Откройте админ-страницу плагина.
3. Нажмите **«Настройки»** и укажите:
   - **Токен пользователя** — строка доступа к API (см. ниже);
   - **Минимальный интервал обновления (сек)** — не чаще одного сохранения на пользователя за этот интервал (по смыслу — защита от дребезга).
4. Сохраните настройки. Плагин перезапустит соединение с новыми параметрами.

Если токен неверный или срок его действия истёк, в индикаторе появится **ошибка авторизации** (HTTP 401/403), а в уведомлениях системы — сообщение об ошибке 2GIS. Обновите токен и сохраните настройки снова.

## Как получить токен

Официальная публичная документация по выдаче **персонального токена Friends** для сторонних приложений может отсутствовать или меняться. На практике токен используется теми же клиентами 2GIS, что и мобильное приложение или веб-интерфейс, когда включён обмен локацией с друзьями.

**Общие правила:**

- Токен привязан к **вашей аккаунтной сессии** в 2GIS. Это чувствительные данные: не публикуйте их в чатах, скриншотах и репозиториях.
- При **смене пароля**, выходе из аккаунта или истечении сессии токен может стать недействительным — нужно получить новый.
- Получение токена обходом официального API приложения может нарушать правила сервиса — используйте на свой страх и риск.

**Типичный способ (веб-браузер, для опытных пользователей):**

1. Войдите в аккаунт 2GIS в браузере (веб-версия карты / личный кабинет, где доступны «Друзья» и геолокация).
2. Откройте **инструменты разработчика** (F12) → вкладка **Сеть (Network)**.
3. Обновите страницу или выполните действие, открывающее карту/друзей.
4. Найдите запросы к хосту **`zond.api.2gis.ru`** или WebSocket **`wss://zond.api.2gis.ru/...`**.
5. В **URL** WebSocket или в заголовках/параметрах запроса найдите параметр **`token=`** — длинная строка после `token=` и до следующего `&` или конца строки — это и есть значение для поля настроек плагина.

**Альтернатива:** перехват HTTPS на своём устройстве с доверенным сертификатом (например, прокси для отладки). Это сложнее и требует понимания безопасности; не используйте на чужих сетях.

Если ничего не находится — убедитесь, что в приложении 2GIS включены **«Друзья»** и **передача геопозиции**, и что вы смотрите именно трафик к `zond.api.2gis.ru`.

## Список друзей и GpsTracker

После успешного подключения список пользователей подтягивается из ответов сервера. В таблице есть колонка **GPSTracker** (иконка в строке): по клику выполняется включение/выключение пересылки данных в **GpsTracker**. Состояние отображается иконкой: галочка — пересылка включена, крест — выключена.

## Технические сведения

При проблемах смотрите журнал приложения (логи модуля Friends2GIS) и состояние индикатора WebSocket на странице плагина.
